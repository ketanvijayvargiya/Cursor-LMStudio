# Cursor Integration Hub: Local LLM (LM Studio) & GitHub Copilot

Welcome to the ultimate guide and resource hub for supercharging your Cursor AI experience. This repository provides a comprehensive walkthrough on how to integrate **LM Studio** (for local LLM inference) and **GitHub Copilot** into the **Cursor IDE**.

By combining these tools, you can enjoy the privacy and cost-effectiveness of local models alongside the powerful suggestions of GitHub Copilot and the agentic capabilities of Cursor.

## 📁 Project Structure

- `docs/`: Contains the main integration article and image resources.
    - `article.md`: The step-by-step guide.
    - `images/`: Descriptions and placeholders for guide screenshots.
- `configs/`: Sample configuration snippets for Cursor and Copilot.
- `scripts/`: Utility scripts to help verify your setup.

## 🚀 Key Features

- **Local LLM Privacy**: Learn how to use LM Studio to run models locally and connect them to Cursor.
- **Cost Efficiency**: Bypass paid API limits by using your own hardware.
- **Copilot Synergy**: Use GitHub Copilot's autocompletion alongside Cursor's Chat and Composer.
- **OpenAI Compatibility**: Leverage the OpenAI-compatible API provided by LM Studio.

## 🛠 Prerequisites

- [Cursor IDE](https://cursor.com/)
- [LM Studio](https://lmstudio.ai/)
- [ngrok](https://ngrok.com/) (optional, for exposing local API to Cursor)
- GitHub Copilot subscription
- Python 3.x (for running utility scripts)

## 📦 Installation

To use the utility scripts, install the necessary Python dependencies:

```bash
pip install -r requirements.txt
```
